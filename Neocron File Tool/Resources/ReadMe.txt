--------------------------------------------------------------------------------
NEOCRON FILE TOOL README


-------------------
WHAT IS IT?
-------------------
The Neocron File Tool (referred to as "nft" from now on) is a tool to compress
and decompress files associated with the game "Neocron 2."


-------------------
SYSTEM REQUIREMENTS
-------------------
Hardware:		Should run on just about anything (Speed will vary by 
			hardware, though)
O/S:			Windows 98 or better
Software:		Visual Basic 6 Runtimes


-------------------
USER REQUIREMENTS
-------------------
A brain
